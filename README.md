# pylogger
A python keylogger. For malicious activity. Possibly

## Features 
- logs keystrokes
- sends emails periodically

## Limitations
- unless cancelled, it keeps sending logs which may result in being blocked by the mail provider
- emails strings, without specifying where the strings are being typed to

## Aim
- to embed the keylogger into something else(image, website or an attachment) and email it to a dummy email account then be able to capture credentails
